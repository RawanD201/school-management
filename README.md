## Setup


```
    cd project
```

```
    cp .env.example .env
```

```
    composer install -o
```

```
    yarn install OR pnpm install
```

```
    php artisan migrate
```

```
    pnpm fresh --seed
    pnpm clear
```

```
    php artisan key:generate
```

```
    php artisan serve & pnpm dev OR yarn dev
```

-   the default username: `admin` and password: `password` for login.
